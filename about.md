# Paimbnails

Paimbnails is a mod focused on improving the Geometry Dash experience with thumbnails, in-game capture, and moderation utilities.

It includes optimized downloading, local caching, and configurable visual effects.

## Main Features

- **Level thumbnails**: previews in lists and related screens.
- **Profile thumbnails**: profile visuals powered by the same thumbnail pipeline.
- **In-game capture (Direct/Render)**: capture flow to generate thumbnails from inside the game.
- **Optimized downloads**: concurrent downloads + local cache to reduce load times and network usage.
- **Visual effects**: animated gradients, particles, hover effects, and configurable styles.
- **Level info screen**: background styles (including “pixel/blur/effects”-style modes).
- **Settings & tools**: UI tweaks, customization options, and performance settings (e.g. Android safe mode).
